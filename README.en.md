# MWebBrowser


#### Introduction

MWebBrowser is a WPF project developed based on CefSharp, mimicking the [Edge](https://microsoft-edge.en.softonic.com/) browser.

CefSharp version has been updated to CefSharp.WinForms.NETCore 120.1.110.
